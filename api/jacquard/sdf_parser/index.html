Module sdf_parser 

Source
Expand description

Parser for Standard Delay Format (SDF) 3.0 files.

SDF files contain per-instance timing delays from place-and-route tools. This parser extracts IOPATH (cell) delays, INTERCONNECT (wire) delays, and TIMINGCHECK (setup/hold) constraints.
