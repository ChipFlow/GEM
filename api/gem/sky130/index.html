SKY130 standard cell library support for GEM.

This module provides pin direction and width information for the SKY130 open-source PDK standard cells. SKY130 cells follow the naming convention: sky130_fd_sc_<variant>__<celltype>_<drive>

Supported variants: hd (high-density), hs (high-speed), ms (medium-speed)
