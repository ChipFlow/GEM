X-propagation-aware CPU partition executor for script version 1.

Mirrors simulate_block_v1 but tracks a parallel X-mask sideband. If the partition is not X-capable (metadata word 8 == 0), delegates to simulate_block_v1 with no overhead.

X-mask semantics: bit=1 means unknown (X), bit=0 means known. For AND gate: ret_x = (a_x | b_eff_x) & (a_eff | a_x) & (b_eff | b_eff_x)
