Module event_buffer 

Source
Expand description

Event buffer for GPU→CPU communication of non-synthesizable constructs.

The event buffer is a shared memory region where the GPU writes events (e.g., $stop, $finish, $display, assertion failures) and the CPU processes them between simulation stages.
